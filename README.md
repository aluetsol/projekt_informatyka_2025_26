Autor: Anastasiia Heraimovych 206470, ARiSS sem.3 - zadanie z informatyki - Arkanoid//projekt C++

Opis: Projekt gry typu Arkanoid wykonany w języku C++ z wykorzystaniem biblioteki SFML.
Gracz steruje paletką, odbija piłkę i niszczy cegiełki o różnej wytrzymałości.

Funkcje gry:
  - sterowanie paletką za pomocą klawisz "A" i "D";
  - piłka, która się odbija od ścian i paletki;
  - cegiełki o różnej wytrzymałości (1–3 trafienia, zależnie od rzędu, ze zmianą koloru po trafieniu);
  - system punktów i ich liczenie;
  - zapis i wczytywanie stanu gry za pomocą klawisza F5;
  - menu główne i ekran instrukcji.

Sterowanie:
  - A / D - sterowanie paletką;
  - strzałka do góry/do dołu - zmiana opcji w menu;
  - Enter - zatwierdzenie wybranej opcji w menu;
  - F5 - zapis stanu gry.

Sposób uruchomienia:
  - otworzyć projekt w Visual Studio;
  - upewnić się, że plik arial.ttf znajduje się w katalogu projektu;
  - skompilować i uruchomić projekt
    
Użyte technologie: C++ oraz SFML;
